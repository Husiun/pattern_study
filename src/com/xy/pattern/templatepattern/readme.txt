设计模式之模版方法模式  Template Method Pattern

模版方法模式是父类调用子类方法的一种设计模式，这种调用不是直接调用，而是通过子类重写基本方法，父类通过模版方法将
基本方法按照一定的顺序，组合起来，模版方法是基类组织基本方法，子类不需要更改；
    具体到本例子中就是run 方法将汽车具有的start,engineBoom,alarm,stop方法按照一定的顺序组织起来，作为模版，而我们
    的实现类有对基本方法做了具体的实现，当客户测试的时候，调用模版方法，而模版方法又是调用具体类的实现方法
    实现了通过基类调用子类
    servlet里面用到了模版方法模式
